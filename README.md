fastSwift is an app that thanks of an SSH server sends, compiles and runs Swift files.

## Features
- [x] Custom servers
- [x] User input
- [ ] Command line arguments
- [x] Multiple files
- [x] UIDocumentBrowserViewController
- [x] Syntax highlighting
